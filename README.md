# cubeplusplus
Template to write C++ programs on STM32Cube's HAL.

### Architecture
```
┌───────────┐         
│Application│         
└┬─┬────────┘         
 │┌▽─────────────────┐
 ││Core (Cube)       │
 │└┬───────────────┬─┘
┌▽─▽──────────────┐│  
│Peripheral Driver││  
└┬────────────────┘│  
┌▽─────────────────▽┐ 
│HAL (Cube)         │ 
└┬──────────────────┘ 
┌▽───────┐            
│Hardware│            
└────────┘ 
```

This translates to the following file system:

```
project directory
 ├─src
 │  ├─application
 │  │  ├─app.cpp
 │  │  ├─app.h
 │  │  └─...
 │  ├─core (cube generated, calls to app added)
 │  │  ├─main.c
 │  │  └─...
 │  └─drivers
 │     ├─example_driver.cpp
 │     ├─example_driver.h
 │     └─...
 └─platformio.ini
```
